# 오늘의 온기 (Warmth of the Day) — 리텐션 시스템 설계

> 작성일: 2026-04-01 | 상태: 설계 완료 (Draft)

## 배경 및 목적

'은둔마을'의 유저들은 대인 관계와 사회적 요구에 지쳐 고립을 택한 경우가 많습니다. 기존의 "매일 글쓰기" 스트릭 시스템은 이들에게 또 다른 '숙제'나 '압박'이 될 위험이 있습니다.

본 설계는 **'성과'가 아닌 '존재'와 '연결'**에 초점을 맞춥니다. 마을에 들러 누군가의 글을 읽거나, 작은 공감(리액션) 하나만 남겨도 그날의 온기가 유지되는 **포용적 리텐션 시스템**을 구축합니다.

---

## 핵심 개념

### 1. 오늘의 온기 (Daily Warmth)
- **기존**: `post_type = 'daily'` 글을 써야만 스트릭 유지.
- **변경**: 아래 활동 중 하나라도 수행하면 그날의 온기(스트릭)가 유지됨.
  - **나누기 (Share)**: 글 작성 (게시글 또는 오늘의 하루)
  - **건네기 (Connect)**: 댓글 작성
  - **머물기 (Stay)**: 다른 유저의 글에 리액션 남기기

### 2. 마음 충전 (Heart Recharge)
- **개념**: 며칠간 접속하지 못했을 때 스트릭이 깨지는 좌절감을 방지하는 '스트릭 방어권'.
- **획득**: 커뮤니티에서 긍정적인 상호작용(리액션을 받음 등)이 쌓이면 자동으로 충전됨.
- **효과**: 미접속 시 자동으로 소모되어 스트릭을 보존하고, 다시 돌아왔을 때 따뜻한 환영 메시지를 표시함.

### 3. 나의 작은 등불 (Personal Lantern)
- **시각화**: 숫자로 된 '연속 며칠' 수치보다, 활동의 강도에 따라 은은하게 밝아지는 '등불' 아이콘으로 표현.
- **밝기 단계**:
  - `Dim`: 활동 없음
  - `Glow`: 리액션/댓글 등 소소한 활동
  - `Radiant`: 글쓰기 등 적극적 활동

## DB 스키마 변경 (Proposed)

### 1. `user_preferences` 확장
```sql
ALTER TABLE user_preferences
  ADD COLUMN IF NOT EXISTS recharge_tokens INT DEFAULT 3, -- 최대 5개
  ADD COLUMN IF NOT EXISTS last_warmth_at DATE; -- 마지막으로 온기를 나눈 날짜 (KST)
```

### 2. 활동 로그 뷰 (Activity Trail)
- 각 테이블의 `created_at`을 UNION하여 '오늘 활동 여부'를 빠르게 판단.

---

## RPC 함수 설계

### 1. `get_my_warmth_status()`
사용자의 오늘 활동 상태와 스트릭 정보를 반환합니다.

- **Input**: 없음 (auth.uid() 사용)
- **Output**:
  ```json
  {
    "has_warmth_today": true,
    "current_streak": 12,
    "recharge_tokens": 2,
    "warmth_level": "radiant",
    "activities": ["post", "reaction"]
  }
  ```

### 2. `recharge_heart_token()`
긍정적 피드백 수신 시 토큰을 충전하는 내부 로직 (트리거 연동).

## 유저 여정 (User Journey)

1. **마을 방문**: 유저가 앱/웹 접속.
2. **소소한 활동**: 기운이 없어 글은 못 쓰지만, 다른 이의 글에 '하트' 리액션 하나를 누름.
3. **온기 확인**: 하단 바의 '등불'이 은은하게 빛나기 시작하며 "오늘의 온기를 나누셨네요!" 알림.
4. **스트릭 유지**: 다음 날 접속 시 "연속 13일째 마을을 밝히고 계시네요."라는 격려 수신.
5. **공백 발생**: 3일간 접속 불가 → 시스템이 `recharge_tokens` 3개를 자동 소모하여 스트릭 보호.
6. **복귀**: "다시 오신 걸 환영해요! 당신이 없는 동안 등불이 당신의 기록을 지켜주었어요." 메시지로 복귀 부담 완화.

## 기대 효과
- **부채감 감소**: '매일 해야 한다'는 압박을 '머무는 것만으로 충분하다'는 위로로 전환.
- **자연스러운 참여**: 글쓰기 문턱이 높은 유저도 리액션을 통해 커뮤니티의 일원임을 실감.
- **리텐션 향상**: '깨진 스트릭' 때문에 앱을 지우는 이탈 경로 차단.
